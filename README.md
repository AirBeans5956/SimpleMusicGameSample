# どうやってダウンロードするの?
右上「Clone or download」と書かれた緑のボタンをクリックし、「Download ZIP」と書かれたリンクをクリックすることで、本サンプルをダウンロードできます。(手慣れてる方はGitでcloneしていただいて構いません。)
